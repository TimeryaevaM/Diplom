Разработка сервиса предсказания стоимости домов на основании истории предложений

Цель:
разработать модель предсказывающую стоимость домов на основании представленного датасета.

Входные данные: 
датасет.

Этапы:
1. Обработка признаков
2. Feature Engineering
3. Preprocess data
4. Применение, оптимизация и сравнение алгоритмов:
	4.1. CatBoostRegressor
	4.2. XGBoost
	4.3. LGBMRegressor
	4.4. RandomForestRegressor
	4.5. KNeighborsRegressor
	4.6. GradientBoostingRegressor
	4.7. TabNet
5. Ансамблирование моделей: Stacking . Сравнение различных моделей
6. Submission

Вывод: Самый лучший результат по значению mape был получен в результате использования модели стекинга.
В качестве результирующей была выбрана модель стекинга с использованием LGBM.
